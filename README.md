# The-Wild-Hare
Проект «The Wild Hare» представляет собой 2D-платформер 

## Описание
Игрок управляет зайцем, который должен пройти уровень, собирая морковки и избегая пчёл.

## Механики
- Передвижение (A/D или стрелки), прыжок (Пробел)
- Сбор предметов (морковки) — увеличивают счёт
- Враги (пчёлы) — уменьшают здоровье при касании
- Система здоровья и поражения
- Условие победы (сбор всех предметов: трех морковок)
## Управление
- A / ← — движение влево
- D / → — движение вправо
- Пробел — прыжок
